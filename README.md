###  核心功能
- **安全登入**：使用 Firebase Auth 確保使用者資料的私密性。
- **即時收支記錄**：隨時新增、刪除帳目，畫面秒速同步。
- **條件過濾**：自動篩選屬於當前使用者的資料，確保隱私。

###  使用技術
- **Frontend**: Next.js (React), Tailwind CSS
- **Backend**: Firebase Authentication, Cloud Firestore
- **Deployment**: Vercel
